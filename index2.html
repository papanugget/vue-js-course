Vue Directives

Directive Purpose Shorthand
v-if Conditional statement
v-else Conditional statement combined with v-if
v-cloak Hides page elements until fully loaded to prevent flashing before VUE.js gets fully loaded
v-for For loop takes in element, index, array used to iterate over arrays
v-on Used for binding event handlers. ex. v-on:click v-on:click = @click
v-bind
